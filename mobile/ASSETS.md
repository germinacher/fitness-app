# Assets Requeridos

Para que la aplicación funcione correctamente, necesitas crear los siguientes archivos en la carpeta `mobile/assets/`:

## Archivos Necesarios

1. **icon.png** (1024x1024 px)
   - Icono de la aplicación
   - Formato: PNG
   - Tamaño: 1024x1024 píxeles
   - Fondo: Transparente o sólido

2. **splash.png** (1284x2778 px recomendado)
   - Pantalla de inicio (splash screen)
   - Formato: PNG
   - Tamaño recomendado: 1284x2778 píxeles (iPhone 11 Pro Max)
   - Puede ser más grande, se ajustará automáticamente

3. **adaptive-icon.png** (1024x1024 px)
   - Icono adaptativo para Android
   - Formato: PNG
   - Tamaño: 1024x1024 píxeles
   - Fondo: Puede ser transparente o sólido

4. **favicon.png** (48x48 px)
   - Favicon para la versión web
   - Formato: PNG
   - Tamaño: 48x48 píxeles

## Estructura de Carpetas

```
mobile/
├── assets/
│   ├── icon.png
│   ├── splash.png
│   ├── adaptive-icon.png
│   └── favicon.png
```

## Herramientas Recomendadas

Puedes usar herramientas online para generar estos assets:
- [App Icon Generator](https://www.appicon.co/)
- [Icon Kitchen](https://icon.kitchen/)
- [MakeAppIcon](https://makeappicon.com/)

## Nota

Si no tienes estos archivos, la aplicación aún funcionará, pero verás errores o imágenes predeterminadas. Es recomendable crear estos assets antes de publicar la aplicación.
